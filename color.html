There are three ways to do color
- Name
- Hex
- Styles using rgba

Name is limited to 140ish defined names that you can lookup

If you use Hex, you have 16 million and change
- For the syntax you are going to use a hashtag/pound sign in front of the colors
- Web safe colors use repeated values

RGBA stands for: Red, Green, Blue, and Alpha.
- Can only be used with style sheets
- Syntax will be: rgba (0-255,0-255,0-255,0-1)
